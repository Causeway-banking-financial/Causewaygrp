# CauseWay Design Reference Notes

## Key Design Elements from Reference Images

### Color Palette (Exact)
- Deep Forest Green: #133129 (primary background)
- Teal: #224B40 (secondary)
- Sage: #406D61 (accents)
- Gold: #d4a84b (highlights, CTAs)
- Cream: #faf9f6 (text on dark, light backgrounds)

### Logo
- Geometric "C" shape with colored squares (green, teal, gold)
- "CauseWay" text in English
- "كوزواي" text in Arabic below

### Typography
- English headings: Playfair Display (serif)
- Arabic headings: Amiri (serif)
- Body text: Cairo (Arabic), Inter (English)

### Key Visual Elements from References
1. Yemen map with data visualization (circuit-board style)
2. Tagline: "For a decade, decisions have been made in the dark. Something is about to change."
3. Observatory section with "Coming Q2 2026" badge
4. Email signup for launch updates
5. Clean navigation with language toggle

### Design Principles
- Asymmetric layouts preferred over centered
- Deep green backgrounds with gold accents
- Professional, institutional feel
- Bilingual excellence (Arabic/English)
- Data visualization elements
- Subtle animations and transitions
