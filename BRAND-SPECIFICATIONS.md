# CauseWay Brand Specifications

## Official Brand Name
- **English**: CauseWay (capital C and W)
- **Arabic**: كوزواي

## Logo Specifications
The CauseWay logo consists of:
1. **Geometric Icon**: A stylized "C" shape made of:
   - Deep forest green outer frame (#133129)
   - Sage green inner square (#406D61)
   - Gold accent square (#d4a84b)
   - Small teal circle accent (#224B40)
2. **Typography**: 
   - "CauseWay" in elegant serif font (white on dark backgrounds)
   - "كوزواي" in Arabic script below (sage green #406D61)

## Primary Color Palette (Forest Green Theme)

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Deep Forest | #133129 | Primary backgrounds, header, footer |
| Teal | #224B40 | Secondary backgrounds, cards |
| Sage | #406D61 | Accents, hover states, text highlights |
| Midnight | #081F1C | Darkest backgrounds, overlays |
| Near Black | #021010 | Text on light backgrounds |

## Accent Colors

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Gold | #d4a84b | CTAs, highlights, premium accents |
| Cream | #faf9f6 | Light backgrounds, text on dark |
| Warm Gold | #c9a54d | Hover states for gold elements |

## Typography

### Headings
- **Font Family**: Playfair Display (serif) or similar elegant serif
- **Weight**: 400-700
- **Style**: Elegant, premium, editorial

### Body Text
- **Font Family**: Inter or similar clean sans-serif
- **Weight**: 400-500
- **Style**: Clean, readable, professional

### Arabic Text
- **Font Family**: Noto Kufi Arabic or similar
- **Direction**: RTL (Right-to-Left)
- **Weight**: 400-600

## Design Elements

### Premium Textures
- Subtle grain/noise overlay on backgrounds
- Gold foil accents on premium elements
- Botanical/leaf motifs for decorative elements
- Marble-like textures for luxury feel

### Shadows and Depth
- Soft shadows for cards and elevated elements
- Subtle gradients from #133129 to #224B40
- Glass morphism effects for overlays

### Borders and Dividers
- Gold (#d4a84b) for premium dividers
- Sage (#406D61) for subtle separators
- 1px borders with low opacity for cards

## Accessibility Requirements
- Minimum contrast ratio: 4.5:1 for text
- Focus states visible on all interactive elements
- Touch targets minimum 44x44px on mobile
- Alt text for all images
